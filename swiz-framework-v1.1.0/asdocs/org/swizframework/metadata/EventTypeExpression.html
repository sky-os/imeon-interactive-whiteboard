parse()method
protected function parse():void

Parse event type expression. Processes an event type expression into an event class and type. Accepts a String specifying either the event type (ex. 'type') or a class constant reference (ex. 'SomeEvent.TYPE'). If a class constant reference is specified, it will be evaluated to obtain its String value. If a ". wildcard is specified, all constants will evaluated. Class constant references are only supported in 'strict' mode.
