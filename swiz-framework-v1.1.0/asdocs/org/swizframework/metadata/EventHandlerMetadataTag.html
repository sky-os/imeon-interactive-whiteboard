eventproperty 
event:String  [read-only]

Returns event attribute of [EventHandler] tag. Refers to the event type that will trigger the decorated method. Is the default attribute, meaning [EventHandler( "someEvent" )] is equivalent to [EventHandler( event="someEvent" )].
